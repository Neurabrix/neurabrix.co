Mobile photo intake

Capture patient intake through a phone photo path from the Dashboard. The short video is designed to explain the feature on its own: what it is, what action to take, and what changes after the action.

Use when Use this when intake details are easier to capture from a phone camera.
Action Click Mobile photo from the same intake toolbar when details are captured at the counter or bedside.
Result The photo-based intake can be reviewed before normal queue handling.
Feature clip: start mobile photo intake from the toolbar and route the captured details for review. Open video

How it works

#

This feature page is intentionally short so the video and the expected result are easy to share.

Feature Mobile photo intake

Capture patient intake through a phone photo path from the Dashboard.

Action What to do

Click Mobile photo from the same intake toolbar when details are captured at the counter or bedside.

Result What changes

The photo-based intake can be reviewed before normal queue handling.

Why it matters

#

Clinics can capture details where the patient is, then route the record into the same visit-day workflow.
